# 📰 Rumor Detection System

A Machine Learning-based web application that detects whether a news article or text is a **Rumor (Fake News)** or **Not a Rumor (Real News)**. The project uses Natural Language Processing (NLP) techniques and a trained machine learning model to classify user-provided text.

---

## 📌 Project Overview

The Rumor Detection System helps identify misleading or fake news by analyzing the input text and predicting whether it is genuine or fake. This project demonstrates the use of **Natural Language Processing (NLP)** and **Machine Learning** for text classification.

---

## ✨ Features

- Detects fake and real news articles
- User-friendly web interface
- Fast prediction using a trained ML model
- Text preprocessing and cleaning
- Machine Learning-based classification
- Easy to use and deploy

## 🚀 How It Works

1. User enters a news article or text.
2. The text is preprocessed.
3. TF-IDF converts the text into numerical features.
4. The trained Machine Learning model predicts the result.
5. The application displays whether the news is **Rumor** or **Not a Rumor**.
